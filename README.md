# Budget Tracker

A simple budget tracking app to manage your income and expenses.

## Features

- ✅ Add income and expenses
- ✅ Categorize transactions
- ✅ View total balance, income, and expenses
- ✅ Delete transactions
- ✅ Clean, responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Usage

1. Enter a description for your transaction
2. Add the amount
3. Select if it's income or expense
4. Choose a category
5. Click "Add Transaction"

Your balance updates automatically!
